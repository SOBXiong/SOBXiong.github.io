top命令

  • 查看整机性能：top(主要cpu)
    • 按数字1查看cpu各核心情况：
      1. us user
      2. sy system
      3. id idle(空闲率,越高越好)
    • load average：a b c(系统1、5、15分钟的系统平均负载量,如果abc平均大于0.6说明系统负担重,大于0.8说明系统快宕机)
    • q退出
  • 低配版：uptime

free命令

  • 查看内存：free(默认kb)
  • free -m(MB)、free -g(GB)

df命令

  • 查看磁盘：df(disk free,默认kb)
  • df -h(以MB为单位)

vmstat命令

  • 查看简单的系统性能：vmstat -n 2 3(2代表每两秒采集一次,3代表共采集三次)
  • 重要参数：
    1. r：runtime process(运行进程数)
    2. b：blocking process(阻塞进程数,越少越好)

iostat命令

  • 查看磁盘IO情况：iostat -xdk 2 3(2,3同上)
  • 重要参数：
    1. r/s：每秒读
    2. w/s：每秒写
    3. %util：一秒中有百分之多少的时间用于I/O操作，或者说一秒中有多少时间I/O队列是非空的
